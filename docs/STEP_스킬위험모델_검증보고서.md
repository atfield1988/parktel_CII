## 1) 위험평가 모델 분석 결과

- 파일: `/home/user/uploaded_files/위험평가 보고서.xlsx`
- 시트
  - `1.개요`: 위험도 계산식 정의
  - `2.위협 목록`: 21개 항목의 위협코드(PARK-01~21) 및 위협도(전 항목 3)
- 적용 계산식
  - 위험도C = 자산중요도C + 취약성 + 위협도
  - 위험도I = 자산중요도I + 취약성 + 위협도
  - 위험도A = 자산중요도A + 취약성 + 위협도
  - 전체 위험도 = 위험도C + 위험도I + 위험도A
  - 취약성 = 항목중요도 × (2 - (평가가중치 × 2))
